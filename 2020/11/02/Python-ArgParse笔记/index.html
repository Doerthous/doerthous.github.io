Python ArgParse笔记

可选参数和位置参数

可选参数由类似add_argument('-f','--foo')的语句创建，add_argument('foo')创建位置参数。当调用parse_args()时，带-前缀的将被识别为可选参数，剩余的被识别为位置参数。

参数配置

通过调用add_argument可设置参数的一些属性，如默认值、参数类型等。

子命令

子命令通过add_subparsers创建，

parser = argparse.ArgumentParser()
subparsers = parser.add_subparsers()
parser_a = subparsers.add_parser('cmd1')
# parser_a.add_argument ...
parser_b = subparsers.add_parser('cmd2')
# parser_b.add_argument ...

用法上，除了在调用时需指定子命令外（如python p.py cmd1 ...），其余基本相同。

一些需求

两种执行模式

  1. python p.py执行一种操作
  2. python p.py -s执行另一种操作

实现

import argparse

parser = argparse.ArgumentParser()
parser.add_argument('-s', action='store_true')
args = parser.parse_args()

if args.s:
    print('python p.py -s')
else:
    print('python p.py')

互斥的多模式

  1. python p.py执行一种操作
  2. python p.py s执行另一种操作
  3. python p.py r执行另一种操作
  4. 1,2,3互斥

实现

import argparse


def func(args):
    print('python p.py')

def func_r(args):
    print('python p.py r')

def func_s(args):
    print('python p.py s')


parser = argparse.ArgumentParser()
subparsers = parser.add_subparsers()

parser_s = subparsers.add_parser('s')
parser_s.set_defaults(func=func_s)

parser_r = subparsers.add_parser('r')
parser_r.set_defaults(func=func_r)

args = parser.parse_args()

if not hasattr(args, 'func'):
    args.func = func

args.func(args)

带参命令

  1. python p.py执行一种操作
  2. python p.py s执行另一种操作
  3. python p.py r执行另一种操作
  4. 1,2,3互斥
  5. python p.py r [-t title]提供可选参数title, 默认为''

实现

import argparse


def func(args):
    print('python p.py')

def func_r(args):
    print('python p.py r title:%s' % args.title)

def func_s(args):
    print('python p.py s')


parser = argparse.ArgumentParser()
subparsers = parser.add_subparsers()

parser_s = subparsers.add_parser('s')
parser_s.set_defaults(func=func_s)

parser_r = subparsers.add_parser('r')
parser_r.set_defaults(func=func_r)
parser_r.add_argument('-t', '--title', type=str, default='', required=False)

args = parser.parse_args()

if not hasattr(args, 'func'):
    args.func = func

args.func(args)
